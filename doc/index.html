Tree控件使用说明

一、属性

1.1 forceImg

是否强制显示图片,优先级最高,0-无操作,1-强制显示所有,2-强制关闭所有

1.2 forceChk

是否强制显示复选框,优先级最高,0-无操作,1-强制显示所有,2-强制关闭所有

1.3 defaultImg

默认的图片,tree-dept

1.4 checkOnlyLeaf

复选框的作用是否是只选中叶子节点,默认false

1.5 expandAll

是否展开所有的节点,默认false

1.6 expandFirst

是否展开第一级节点,默认false

1.7 expandSelected

是否展开所有选中的节点,默认true

1.8 selectRow

是否对选中的节点行上色,默认true

1.9 hoverRow

是否在鼠标悬停到节点时高亮显示,默认true

1.10 nodeClickType

单击行文本时是展开子节点还是触发事件还是选中节点("expand"|"click"|"check"),默认expand

1.11 nodeClick

单击文本的点击事件,当textClickType的值为"click"时有效

1.12 beforeCheck

选中节点前触发

1.13 beforeUnCheck

取消选中节点前触发

1.14 afterCheck

选中节点或取消训中节点后触发

1.15 animate

在节点展开或收缩时是否显示动画效果,默认false

1.16 field_Text

显示节点名称的字段名称,默认Text

1.17 field_Open

决定节点是否展开或收缩的字段名称,默认Open

1.18 field_ImgCls

决定节点显示的是哪个图标的字段名称,默认ImgCls

1.19 field_ChildNodes

决定节点的字节点的字段名称,默认ChildNodes

1.20 field_Checked

决定节点是否处于选中状态的字段名称,默认Checked

1.21 field_ShowImg

决定节点是否显示图片的字段名称,默认ShowImg

1.22 field_ShowCheckBox

决定节点是否显示复选框的字段名称,默认ShowCheckBox

                        {

                          "ShowImg": true,
